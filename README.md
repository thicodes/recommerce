## Run locally (Develop)

### Run the Recommerce Server

#### Install dependencies

```bash
yarn install
```

#### Install dependencies

```bash
yarn watch
```

### Run the Recommerce Client

#### Install dependencies

```bash
yarn install
```

#### Install dependencies

```bash
yarn start
```
